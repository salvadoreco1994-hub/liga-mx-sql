# Liga MX SQL

Este repositorio contiene un proyecto de análisis de resultados de la **Liga MX** usando **SQL Server**.  
El objetivo es importar datos de partidos, almacenar cuotas (momios) y realizar consultas estadísticas.

---

## 📂 Estructura del proyecto

- `/data` → Archivos CSV con los partidos de la Liga MX.
- `/sql` → Scripts SQL para crear y modificar tablas.
- `/analysis` → Consultas de resultados y reportes.
- `README.md` → Documentación del proyecto.
